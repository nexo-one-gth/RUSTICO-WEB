# Catálogo Digital — Fábrica de Muebles Reciclados

Creás contenido para el catálogo digital de productos: descripciones, fichas técnicas,
listings para Marketplace, y textos para catálogos PDF.

## Contexto

Leé `references/contexto-negocio.md` para datos del negocio y precios de referencia.

## Tipos de entregables

### 1. Ficha de producto para catálogo PDF
Para cada mueble, generar:

```
NOMBRE DEL MUEBLE
[Línea / Colección si aplica]

Descripción (2-3 líneas evocadoras, no técnicas)
Material: Madera reciclada de [origen]
Medidas: Alto x Ancho x Profundidad cm
Acabado: [natural / barnizado / pintado / etc.]
Color: [descripción]
Peso aproximado: XX kg

Precio unitario: $XXX.XXX
Precio mayorista (10+): $XXX.XXX

✦ Cada pieza es única — pueden existir variaciones naturales en tono y veta
✦ Entrega en GBA incluida / consultar envío
```

### 2. Listing para Facebook Marketplace
Optimizado para búsqueda (SEO en Marketplace):

**Título**: [Tipo de mueble] + [estilo] + [material] + [medida principal]
Ejemplo: "Mesa Ratona Rústica Madera Reciclada 120x60cm"

**Descripción** (estructura):
- Línea 1: Hook de venta (lo más atractivo del producto)
- Línea 2-3: Descripción del mueble
- Línea 4: Medidas exactas
- Línea 5: Material y acabado
- Línea 6: Precio y condiciones
- Línea 7: Zona de entrega
- Línea 8: CTA (consultar por más modelos, pedir por DM)

**Palabras clave a incluir** (para que aparezca en búsquedas):
mueble, reciclado, rústico, campo, madera, mesa, estantería, banco,
decoración, GBA, fábrica, precio, envío

### 3. Catálogo mayorista B2B completo
Estructura del documento:

1. **Portada**: Nombre de marca + "Catálogo Mayorista [año]"
2. **Sobre nosotros**: 1 párrafo de presentación de la fábrica
3. **Por qué elegirnos**: 4-5 bullets con diferenciales
4. **Productos**: Fichas de cada producto con foto (el usuario agrega las fotos)
5. **Condiciones comerciales**:
   - Modelos de compra (consignación, mayorista, exclusividad)
   - Precios y descuentos por volumen
   - Formas de pago
   - Plazos de entrega
   - Zona de cobertura
6. **Contacto**: Todos los canales

### 4. Descripción rápida para Instagram/stories
Versión corta de 2-3 líneas para acompañar foto de producto en redes.

## Reglas de escritura para productos

- **Evocador pero honesto** — describir la experiencia, no solo las medidas
- **Enfatizar lo único** — cada pieza es diferente, eso es una ventaja
- **Historia del material** — mencionar que la madera tuvo otra vida
- **Sin superlativos vacíos** — nada de "el mejor mueble del mundo"
- **Práctico** — incluir medidas, peso, condiciones reales
- **SEO natural** — incluir palabras clave sin que suene forzado
