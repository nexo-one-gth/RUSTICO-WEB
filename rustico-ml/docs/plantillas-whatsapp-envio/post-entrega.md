# Plantilla: Post-Entrega

## Mensaje Inmediato Post-Entrega (mismo día)

```
Hola [NOMBRE]! Espero que hayas llegado bien a casa 🏡

¿Pudieron acomodar el [PRODUCTO] donde querían?
¡Nos encantaría ver cómo quedó en el espacio! 📸

Si tenés cualquier consulta o necesitás algo,
acá estamos 🪵

— Nexo, Rústico
```

---

## Mensaje Pedir Reseña en ML (2–3 días después)

```
Hola [NOMBRE]! Cómo te fue con el [PRODUCTO]? 🙌

Te pido un favorcito: si podés dejarnos
una calificación en Mercado Libre, nos ayuda
un montón a seguir creciendo 🌟

Solo tardás un minutito y para nosotros
significa todo en este inicio.

¡Muchas gracias de corazón! 🪵
— Nexo, Rústico
```

---

## Mensaje con Pedido de Foto (para redes)

```
Hola [NOMBRE]!

¿Te animás a mandarnos una foto de cómo quedó
el [PRODUCTO] en tu casa? 📸

Con tu permiso nos gustaría compartirla
en nuestro Instagram para mostrar el mueble
en un espacio real 🪵

No es obligatorio, ¡pero si lo hacés te amamos! ❤️
```

---

## Mensaje si Hay Problema Post-Entrega

```
Hola [NOMBRE], gracias por avisarme.

Entiendo tu molestia y lo lamento.
Decime bien qué está pasando para
ver la mejor solución 🙏

Trabajamos artesanalmente y siempre
queremos que cada cliente quede 100% conforme.

¿Podés mandarme una foto para ver bien
el detalle? Así lo resolvemos rápido.
— Nexo
```
