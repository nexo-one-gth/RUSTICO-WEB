# Plantilla: Coordinar Horario de Entrega

## Propuesta de Fecha

```
Hola [NOMBRE]! 🙌

Tu mueble está listo para entrega.

📅 Puedo llevarlo el [DÍA] [FECHA] en el siguiente horario:
⏰ Mañana: 9:00 a 12:00hs
⏰ Tarde: 14:00 a 18:00hs

¿Cuál te viene mejor?

Dirección confirmada: [DIRECCIÓN QUE DIO EL CLIENTE]
¿Está bien esa dirección?
```

---

## Recordatorio el Día Anterior

```
Hola [NOMBRE]! Te recuerdo que mañana [DÍA] [FECHA]
llevo tu [PRODUCTO] entre las [HORA INICIO] y [HORA FIN]hs.

📍 Dirección: [DIRECCIÓN]

¿Confirmamos? 🪵
```

---

## Si el Cliente No Responde (24hs después)

```
Hola [NOMBRE]! Te escribo de Rústico, quería coordinar
la entrega de tu pedido.

¿Cuándo te viene mejor para recibirlo?
Tengo disponibilidad [DÍAS DISPONIBLES].

¡Avisame cuando puedas! 🪵
```
