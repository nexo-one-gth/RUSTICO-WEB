# Scripts de Upsell — Subir el Ticket desde una Pregunta

> El momento en que alguien te pregunta es el momento de mayor interés.
> Usá estos scripts para subir el valor del pedido de forma natural.

---

## UPSELL 1: De pieza individual al Combo

**Situación:** El cliente pregunta solo por el Rack TV

```
¡Hola! El Rack TV está disponible a $[PRECIO].

Una cosa que muchos clientes terminan eligiendo
es el Combo Living completo, que incluye también
la Mesa Ratona y la Mesa de Arrime.

Las 3 piezas están diseñadas juntas, así que
combinan perfecto. Y el combo tiene mejor precio
que las 3 piezas por separado.

¿Querés que te pase el precio del combo para que compares? 🙌
```

---

## UPSELL 2: Agregar personalización pagada

**Situación:** El cliente está por confirmar

```
¡Perfecto! Antes de confirmar, te cuento
que podemos hacerlo con algunas opciones adicionales:

➕ Estante extra en el Rack: $[PRECIO]
➕ Rueditas en la Mesa Ratona: $[PRECIO]
➕ Acabado especial en la madera: $[PRECIO]

¿Te suma alguna de estas opciones?
Si no, también lo hacemos tal como está en la publicación 🙌
```

## UPSELL 3: Segundo mueble para otro ambiente

**Situación:** Post-venta, mensaje de seguimiento

```
Hola [NOMBRE]! Cómo quedó el living? 🙌

Muchos clientes que empiezan con el Combo Living
después nos piden muebles para otros ambientes.

Si en algún momento necesitás algo para el comedor,
dormitorio, o una pieza particular, avisanos.
Como ya te conocemos, te damos prioridad y
un precio especial 🪵

¡Saludos!
```

---

## DOWNSELL: Si el precio es el obstáculo

**Situación:** El cliente dice que le queda caro

```
Te entiendo. ¿Qué presupuesto tenés en mente?

Si el combo completo no entra por ahora,
podemos empezar solo con el Rack TV
y después sumar las mesas cuando puedas.

O también podemos hacer un plan de pago
en cuotas a través de Mercado Libre 💳

¿Cuál es tu tope de presupuesto?
```
